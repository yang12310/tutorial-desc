01 포트폴리오 개요

  • U-Jeans는 Front-End 개발자 취업을 위한 Portfolio로 개발된 SNS(Social Network System) 시스템입니다.

  • U-Jeans는 Figma를 이용한 UI 디자인과, Client Side는 React, Redux-Toolkit으로 기능을 구현하였고, Server Side는 Node.js, JWT로 기능을 구현하였으며, Database는 MongoDB를 사용한 MERN Full-Stack 시스템입니다.

  • 주요 기능으로는 회원가입, 로그인, Post등록 및 수정, 삭제기능과 친구추가, 삭제기능 등이 있습니다.

  • Server는 AWS Lightsail Cloud에서 서비스되고 있으며, Client는 Github에 배포하였습니다.

  • 참고로 본 포트폴리오인 U-Jeans 브랜드 의미는 도전과 개척, 진취성을 의미합니다.

02 시스템구성 및 적용기술

U-Jeans 개발환경은 다음과 같습니다.

  • 01개발플랫폼(Hardware)

    MacBook Air Retina, macOS Sonoma 14.5, 8G

  • 02디자인 (Figma)

    웹디자인 2년 경력을 바탕으로 Figma로 직접 디자인하였습니다.

  • 03Front-End

    HTML / CSS / JavaScript 및 React, Redux-Toolkit, Formik등으로 Client측 기능을 구현하였습니다.

  • 04Back-End

    Node.js환경에서 JWT를 적용하여 Server측 기능을 구현하였습니다.

  • 05Database

    Database는 MongoDB를 사용하였습니다.

04 주요기능 설명

  • 1.사용자 등록

    이름(First Name, Last Name), 지역(Location), 직업(Occupation), 사용자이미지(Picture), 이메일(Email), 비밀번호(Password)를 입력합니다.

  • 2.로그인 / 로그아웃

    이메일(Email), 비밀번호(Password)를 입력합니다.

    로그아웃 메뉴를 선택합니다.

  • 3.Home

    로그인하면 Home 화면이 Display됩니다.

    Home 화면에서는 Post 전체보기 ,Mypage 이동, 친구추가, 친구삭제, 친구페이지 방문하기 등의 기능이 있습니다.

    모든 사용자가 등록한 Post를 볼 수 있습니다.

  • 4.Mypage 이동

    내가 작성한 Post를 확인, 등록, 수정, 삭제할 수 있습니다.

    등록된 사용자 중 친구로 등록하거나 등록된 친구를 삭제할 수 있습니다.

    친구 페이지를 방문하여 친구가 작성한 Post를 볼 수 있습니다.
